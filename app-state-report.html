Releasu
AI-powered aplikace pro vyuku anglictiny s Gemini. Podporuje real-time konverzaci (Live API), 14 role-play scenaru, session progress tracking, CEFR level assessment a nastavitelnou uroven korekci (1-10).

Silne stranky

  • Gemini Live API: real-time audio konverzace
  • 14 role-play scenaru pro prakticke situace
  • Nastavitelna strictness (1-10) s 5 urovnemi
  • Session summary (silne stranky, chyby, cviceni)
  • Lesson history + streak tracking + trend
  • Premiovy UI (Avatar mod, animace, dark theme)
  • PWA-ready (manifest, install banner)
  • CZ-first: ceske instrukce a preklady

Slabe stranky

  • App.tsx: 844 radku (monoliticke, tezko udrzovatelne)
  • API klic ve frontendu (bezpecnostni riziko)
  • Deprecated ScriptProcessorNode (Live audio)
  • Zadne testy (unit, integration, e2e)
  • TTS voice selection zavisle na OS
  • Zadna serverova perzistence (jen localStorage)
