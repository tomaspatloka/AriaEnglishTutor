Přehled aplikace

Aria je React/Vite PWA s Gemini Live API, legacy STT/TTS módem, scénáři, progressem a samostatným Reading Mode.

v1.4.4 podle blueprint.md
Co je zdravé: oddělené komponenty, kvalitní prompt pro výslovnost, Live API pipeline, localStorage pro rychlý osobní provoz, dobrý vizuální feedback hlasu.
Co limituje výuku: Reading Mode nemá vlastní text knihy, neumí uložit české překlady slov a nenapojí své session do progress historie.
